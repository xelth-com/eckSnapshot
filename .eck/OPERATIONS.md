# Common Operations

## Development Setup
```bash
# Setup commands
npm install
# or yarn install
```

## Running the Project
```bash
# Development mode
npm run dev

# Production build
npm run build
```

## Testing
```bash
# Run tests
npm test

# Run tests in watch mode
npm run test:watch
```

## Deployment
```bash
# Deployment commands
npm run deploy
```
